🍽 FeedMill Restaurant Website

A modern restaurant ordering interface built with HTML, Tailwind CSS, and JavaScript.
This project demonstrates a responsive UI for browsing menu items and placing food orders online.

---

🚀 Features

- Responsive navigation bar
- Mobile sidebar menu
- Restaurant hero section
- Food menu display
- Cart icon with item counter
- Checkout page
- Login page
- Footer with social links
- Mobile friendly design

---

🛠 Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome Icons
